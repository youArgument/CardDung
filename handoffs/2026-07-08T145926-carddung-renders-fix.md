# Handoff: CardDung — Fix Full Grid Re-render After Every Action

## Цель
Устранить видимую «мигалку»/перерисовку игрового поля после каждого действия в dungeon. Игрок кликает карту → весь DOM перестраивается → экран мерцает.

## Сделано
- `js/main.js:573` — GridUI.render() → GridUI.updateCell(cell) при сборе предмета (только одна ячейка обновляется)
- `js/main.js:624, 633` — убраны лишние GridUI.render() после ревила карт. renderRevealedCard() уже отрисовывает ячейку на месте через DOM manipulation
- `js/main.js:1264` — firstTurn врага: GridUI.render() → GridUI.updateCell(cell). firstTurn не меняет визуальное состояние поля
- `js/main.js:1287-1293` — вынесен GridUI.render() из for-цикла enemyAITick. Было: N врагов = N полных перестроений DOM. Стало: updateCell() в цикле + 1× render() после всех врагов
- `js/main.js:672` — убран HandUI.render() при провале игры карты (рука не изменилась)
- Задеплоено через Docker Compose на https://game.you-argument.ru, версия 0.3.1
